Simple Fullstack project
=====================

A starting skeleton project for Django and Angular + material frameworks

## 1. Installing project

```bash
git clone project_name
```

```bash
cd project_name
```

```bash
git submodule init
```

```bash
git submodule update
```

## 2. Installing Client

```bash
cd project 
```   

```bash
npm install
``` 

```bash
npm start  
``` 

## 3. Installing Server
  
```bash
cd server  
```  
   
```bash
pipenv install
```

```bash
pipenv shell
```

```bash
./manage.py migrate
```

```bash
./manage.py createsuperuser
```

```bash
./manage.py runserver
```
